## Positive Test Cases
Acceptance Criteria: User receives reset link
Test case Title: Verify Successful Password Reset Link Generation
Steps:
  1. Navigate to the login page.
  2. Click on "Forgot Password" link.
  3. Enter a valid email address and click on "Reset Password".
Expected Result: A reset link is sent to the provided email address.

Acceptance Criteria: Link expires in 24 hours
Test case Title: Verify Expiration of Reset Link After 24 Hours
Steps:
  1. Request a password reset and receive the link immediately.
  2. Wait for 25 hours without opening the link.
  3. Click on the received link.
Expected Result: The link is invalid or redirects to an error page.

Acceptance Criteria: New password must be 8 chars long
Test case Title: Verify Valid Password Length (Minimum 8 characters)
Steps:
  1. Request a password reset and receive the link immediately.
  2. Click on the received link.
  3. Enter a new password that is exactly 8 characters long and click "Reset Password".
Expected Result: The password is successfully updated, and the user is logged in with the new password.

## Negative Test Cases
Test case Title: Validate scenario behaves as expected
Steps:
  1. Navigate to the login page.
  2. Click on "Forgot Password" link.
  3. Leave the email field empty and click on "Reset Password".
Expected Result: An error message is displayed, preventing the password reset request.

Test case Title: Validate scenario behaves as expected
Steps:
  1. Navigate to the login page.
  2. Click on "Forgot Password" link.
  3. Enter an incorrect email address and click on "Reset Password".
Expected Result: An error message is displayed, preventing the password reset request.

Test case Title: Validate scenario behaves as expected
Steps:
  1. Request a password reset and receive the link immediately.
  2. Click on the received link.
  3. Enter a new password that is less than 8 characters long and click "Reset Password".
Expected Result: An error message is displayed, preventing the password update due to the short length of the entered password.
